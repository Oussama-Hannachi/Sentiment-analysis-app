# Sentiment Analysis App (LLM + Streamlit)

This project is a simple **Sentiment Analysis Web App** built using **Python, Streamlit, and Google's Gemini API**.

The application allows users to input text and automatically detect whether the sentiment is **Positive**, **Negative**, or **Neutral** using a **Large Language Model (LLM)**.

This mini-project was developed as part of a lesson on **LLM Chains and Runnables**.

---

## Features

- Simple web interface using **Streamlit**
- Sentiment analysis powered by **Google Gemini API**
- Real-time sentiment detection
- Lightweight and easy to run locally
- Demonstrates basic **LLM prompt engineering**

---

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- LLM Prompting
